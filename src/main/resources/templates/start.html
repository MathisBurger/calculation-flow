Calculation flow

Welcome to Calculation flow. This application helps you to implement customizable calculations for your standard software.
You can deeply integrate our API endpoint into your application while you can easily configure how specific calculation should be done.
If you want to start using this application just modify the application.yml file for your needs. It contains all calculation functions that
you want the application to perform. After registering them correctly in the config file, you have to restart the server.
After restarting you can start configuring your workflows under the landing tab.
